FAQ

Common questions.

Do I need coding skills to use this?
No. Open the HTML file in your browser, paste your Claude API key, and everything works through a click-based interface. No code, no server, no setup.
What is a Claude API key and where do I get one?
Claude is Anthropic's AI. You get an API key at console.anthropic.com — it costs roughly $0.003 per compliance audit. Most users spend under $5/month on API usage.
Is my data safe? Where is it stored?
Everything stays in your browser's local storage. Your API keys and credentials never touch any server except the original provider (e.g. Anthropic for AI calls). Nothing is stored externally.
Can I use this on multiple devices or share with my team?
You can open the file on any device. Since data is stored locally per browser, each device starts fresh. For team use, the Managed Setup is better as I configure a shared version.
What's included in the 3 custom policies for Managed Setup?
You choose any 3 from: Data Retention, Access Control, Incident Response, AI Usage, Vendor Risk, API Security, Data Classification, Business Continuity, Acceptable Use, or Password Policy — all customized to your org.
Is there a subscription or recurring fee?
No. Both products are one-time purchases with lifetime access. The only ongoing cost is your Anthropic API usage, which you control entirely.
What if I need ongoing monthly compliance monitoring?
After your Managed Setup delivery, you can ask about the monthly retainer option ($197/mo) where I run your audit, update your risk score, and deliver a compliance report every month.
How long does Managed Setup take?
48 hours from when I receive your completed onboarding form. The form comes in your confirmation PDF immediately after purchase. Most clients receive delivery well within 24 hours.
